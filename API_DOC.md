**Api for server listing**
----
* **URL**

  https://servers-list-demo.herokuapp.com/api/v1/servers-list?location=2&hdd=&ram=4,8,12&storage=250,1000&page=1

* **Method:**
  
    `GET`
 
*  **URL Params**


   **Mandatory:**
   `page=1`
   `location=1`
   `hdd=2`
   `ram=2,4,8,12`
   `storage=250,500`

* **Success Response:**
  
    **Code:** `200` 
    **Content:** 
